# The Prose & The Python 🐍 ✒️

> *Where syntax meets satire.*

Welcome to the repository for my personal website and blog, **The Prose & The Python**. This site serves as a digital garden where I document my journey learning Python, share coding tutorials, and publish satirical takes on modern life and corporate culture.

## 🚀 About This Project

This site is hosted on [GitHub Pages](https://pages.github.com/) and serves two main purposes:
1.  **The Python:** Documenting my progress as I learn Python, from "Hello World" to building my first programs.
2.  **The Prose:** A home for my creative writing, featuring satirical essays on everything from the "Iliad of Integration" to the state of city roads.

## 🛠️ Tech Stack

* **Host:** GitHub Pages
* **Generator:** Jekyll (Default GitHub Pages runner) / Markdown
* **Theme:** Minimal

## 📂 Structure

* `index.md`: The homepage and main landing area.
* `_posts/`: (Planned) Directory for blog entries.
